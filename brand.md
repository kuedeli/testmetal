## 8. Design Direction Recommendations

Based on the reference sites and metal-works.ch's positioning:

### Visual Direction
| Aspect | Current | Target Direction |
|--------|---------|-----------------|
| **Background** | White, basic Wix | Dark/moody (charcoal, anthracite) with warm fire accents |
| **Typography** | Default sans-serif | Premium sans-serif (e.g. Outfit, Inter, or DM Sans) with strong headings |
| **Imagery** | Small thumbnails, grid | Full-bleed hero sections, dramatic fire photography |
| **Layout** | Stacked Wix sections | Modern edge-to-edge sections with alternating rhythms |
| **Navigation** | Simple text menu | Sticky header with clean product categories |
| **Products** | Text-heavy specs lists | Visual product cards with pricing overlays |
| **CTA** | Plain text links | Bold contrast buttons with hover animations |
| **Mobile** | Basic Wix responsive | Mobile-first design with touch-friendly product browsing |

### Recommended Color Palette
```
--color-bg-primary:    #1A1A1A;  /* Deep charcoal background */
--color-bg-secondary:  #2D2D2D;  /* Slightly lighter sections */
--color-bg-card:       #333333;  /* Card backgrounds */
--color-text-primary:  #F5F5F0;  /* Warm off-white text */
--color-text-secondary: #A0A0A0; /* Muted descriptions */
--color-accent:        #E8630A;  /* Fire orange (from product imagery) */
--color-accent-warm:   #C44B00;  /* Deeper fire red-orange */
--color-accent-gold:   #D4A853;  /* Warm gold for premium feel */
--color-steel:         #8B8E94;  /* Cool steel grey */
--color-rust:          #B35A25;  /* Rust patina accent */
```

### Recommended Typography
```
--font-heading: 'Outfit', sans-serif;       /* Bold, modern, clean */
--font-body: 'Inter', sans-serif;            /* Excellent readability */
--font-accent: 'DM Serif Display', serif;    /* For dramatic product names */
```

### Key Design Patterns to Adopt

1. **From Creasteel**: 
   - Product feature grid with icons
   - Customer review integration
   - Clean e-commerce product presentation
   - "Swiss Quality" trust badges
   
2. **From Lionfire**:
   - Dark, moody photography style
   - Product line naming/branding
   - Dramatic hero sections
   - Artisan storytelling narrative
   - Workshop/exhibition integration

3. **Metal-works.ch unique advantages to highlight**:
   - Engineer + metalworker background → "Engineered Perfection"
   - Patent holder → trust & innovation signal
   - 6mm steel thickness → "Thicker than the competition" comparison
   - Custom work → configurator/quote request flow
   - Swiss-made handcraft → quality positioning
